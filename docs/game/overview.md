# 军团冲锋 — 游戏概述

## 一句话

竖屏自动前进生存游戏，主角带领一队英雄不断前进，用三选一构建阵容，20 波后击退海量敌人。

## 核心循环

```
场景滚动前进 → 敌人从上方生成 → 我方单位自动攻击
    ↑                                      ↓
    三选一升级 ← 杀光一波获得经验 ← 积累经验/杀敌
```

## 屏幕布局

```
┌──────────────┐
│  敌人刷新区   │  ← 上方不断生成敌人
│              │
│  ●  ●  ●     │  ← 敌人向下移动
│              │
│  [英][主][英] │  ← 我方单位在画面中下部
│  [英] [英]   │
│              │
│   ↓ 场景滚动  │  ← 背景持续向下移动
├──────────────┤
│  HP条 | 波数  │  ← HUD
└──────────────┘
```

竖屏比例 ~9:16。场景宽度固定为 390px，高度自适应。

## 胜利与失败

- **胜利**：消灭全部 20 波敌人
- **失败**：我方所有单位 HP 归零

## 核心机制

1. **自动前进**：场景持续向下滚动，角色保持画面相对位置
2. **自动战斗**：我方单位自动索敌并释放技能
3. **三选一**：每杀光一波敌人触发一次三选一
4. **单位收集**：三选一抽取新单位 or 已有单位的升级
5. **单位血量**：每个单位有独立 HP，归零则阵亡

## 三选一规则

- 每波敌人杀光 → 获得一次三选一机会
- 卡池包含：
  - **新单位**：尚未上场的英雄（已上场的不会再次出现）
  - **单位升级**：针对已有单位的强化
- 三张卡牌展示，点击选择一张
- 如果所有单位都已上场且抽到新单位，返回 unit upgrade 池

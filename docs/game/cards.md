# 军团冲锋 — 三选一卡牌系统

## 触发条件

每波敌人全部消灭 → 弹出三选一界面 → 选择 → 下一波开始

## 卡池生成逻辑

```
本轮可选的卡 = 空闲新单位池 ∪ 已有单位的升级池
```

### 新单位池
- 初始包含所有 7 个单位（除主角）
- 如果某个单位已经在场上，从池中移除
- 不会再次出现

### 升级池
- 为场上每个单位生成对应的升级选项
- 每个单位有多个升级方向，已选过的升级不再出现

### 生成规则

1. 收集所有可选的卡（空闲新单位 + 可用升级）
2. 随机选取 3 张
3. 如果可选项不够 3 张 → 从已有升级中补足（允许重复方向）

## 升级类型

详见各单位文档。通用升级模板：

| 升级名 | 效果 | 适用单位 |
|--------|------|---------|
| 伤害提升 | 攻击力 +X% | 所有 |
| 攻速提升 | 攻击速度 +X% | 所有 |
| 技能增强 | 技能特定参数提升 | 特定 |

## UI 交互

- 游戏暂停（场景停止）
- 展示 3 张卡牌（竖排或横排）
- 点击卡牌选择 → 应用效果 → 继续游戏

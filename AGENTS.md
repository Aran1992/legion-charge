# AGENTS.md - 军团冲锋 (Legion Charge)

## 项目定位

竖屏 PixiJS 自动前进生存游戏 — 控制一队英雄不断前进，杀敌升级选卡，20 波后击败海量敌人获胜。

## 技术栈

- Vite + TypeScript + PixiJS
- 竖屏布局（mobile-first, 9:16）

## 仓库

- **GitHub:** `https://github.com/Aran1992/legion-charge.git`

## 协作规则

1. **文档优先**：玩法/数值变更先改 `docs/game/`，再改 `src/`
2. **提交规范**：每次功能完成后审查并 Git 提交
3. **推送**：审查 → Git commit → push 到 GitHub master

## 游戏流程

1. 场景自动向下滚动（竖屏）
2. 主角自动前进在场景中
3. 敌人从上方生成，向下移动
4. 我方单位自动攻击敌人
5. 杀敌积累经验，杀光一波 = 一次三选一
6. 三选一出现新单位/升级
7. 20 波，每波敌人递增，最后海量
8. 全灭敌人 = 胜利；我方全部单位阵亡 = 失败
